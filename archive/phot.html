Asteroid Photometry and Lightcurves

These data sets contain Photometry observations, which are used to measure the brightness of celestial objects. These measurements can provide information about an objectâ€™s size, distance, temperature, and rotational period. These data sets are mostly asteroid data sets with some comets and other solar system bodies.
