OSIRIS-REx Raw Radio Science Bundle

This bundle collects Tracking and Navigation Files (TNF), Ionospheric Data Files (ION), Troposphere Data Files (TRO), and Weather Data Files (WEA) acquired by the Origins, Spectral Interpretation, Resource Identification, Security, Regolith Explorer (OSIRIS-REx) spacecraft and the Deep Space Network (DSN) on Earth.

Data now available for Bennu encounter through May 17, 2020. For those who want the original DSN versions of the TRK-2-34 files, they are available here.

Release dates:

  • Initial release Aug. 10, 2020: Bennu encounter data through Nov. 17, 2019
  • Release Aug. 25, 2020: Data for Nov. 18, 2019 - Feb. 17, 2020
  • Release Dec. 1, 2020: Data for Feb. 17 - May 17, 2020

These data are certified and released with liens. Please read the liens list before using the data. Note that these sub-bundles will be merged prior to final archiving.
