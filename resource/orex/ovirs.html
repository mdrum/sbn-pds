OSIRIS-REx OVIRS Bundle

This bundle collects all the operational data products produced by the OSIRIS-REx Visible and InfraRed Spectrometer (OVIRS). OVIRS is used for the spectral characterization of the surface of (101955) Bennu.

The archived bundle now includes all data through Aug. 17, 2020.

  • Initial release on Feb. 15, 2019: Outbound Cruise (including Earth/Moon)
  • Release on Aug. 19, 2019: Bennu data through Nov. 17, 2018
  • Release on Aug. 27, 2019: Bennu data through Feb. 17, 2019
  • Release on Nov. 25, 2019: Bennu data through May 17, 2019
  • Release on Feb. 25, 2020: Bennu data through Aug. 17, 2019
  • Release on July 23, 2020: Bennu data for Aug. 17 - Nov. 17, 2019
  • Release on Aug. 25, 2020: Bennu data for Nov. 18, 2019 - Feb. 17, 2020
  • Release on Dec. 1, 2020: Bennu data through May 17, 2020
  • Release on Mar. 5, 2021: Bennu data through Aug. 17, 2020
